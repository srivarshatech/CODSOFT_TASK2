# CODSOFT TASK 3 - Password Generator

## Objective

Develop a Python application that generates strong and customizable passwords based on user preferences.

## Features

- Custom password length
- Lowercase letters
- Uppercase letters
- Numbers
- Special characters
- Input validation
- Generate multiple passwords

## Technologies Used

- Python 3
- random module
- string module

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Run the following command:

```bash
python password_generator.py
```
